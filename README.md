# HRGC-AI
Highway–rail grade crossing (HRGC) crashes often result in severe injuries and fatalities, yet existing HRGC severity-prediction studies rely primarily on structured data and are rarely evaluated under temporally ordered designs. This study develops and evaluates a multimodal HRGC crash-severity framework using over 30,000 nationwide crash records from 2010 to 2025 and examines the added value of narrative text under both random and chronological evaluation designs. Structured, text, and combined models were trained using several machine-learning algorithms and assessed under both train–validation–test splits to isolate the influence of temporal ordering, while explainable artificial intelligence was used to interpret the contribution of key predictors. Combined models consistently outperformed single-source models, and the combined XGBoost classifier achieved the strongest temporal performance, including a balanced accuracy of 0.75 and notable gains in identifying fatal crashes. Text models also outperformed all structured models, highlighting the predictive value of narrative descriptions in HRGC severity prediction. Feature contribution analysis identified highway user type, highway user age, train speed, highway user position, visibility, and roadway condition as key predictors. These findings show that leveraging multimodal data provides agencies with more reliable tools for identifying high-risk locations and prioritizing safety interventions.
